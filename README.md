## Game Overview

Jet Strike Max Intense is a browser-based survival shooter that drops you into a steadily intensifying war against wave after wave of alien attackers. As each round advances, enemy formations grow harder, denser, and more aggressive, so fast reactions and careful power-up timing matter just as much as accuracy. The gameplay loop stays focused: stay alive, grab upgrades, and keep pushing deeper into the attack pattern.

What sets the game apart is the rising difficulty ladder that builds toward the famous 100th wave. Power-ups appear throughout the run and can give you short-lived boosts such as stronger weapons, shields, or spread fire, all of which can rescue a run on the edge of collapse. The 2026 update puts emphasis on tuning those power-up effects and smoothing out wave progression so the challenge remains demanding without feeling unfair.

## Options

A small configuration block sits near the top of the script. The main settings are:

| Setting | Description | Default |
|---------|-------------|---------|
| `initialLives` | Starting lives count | 3 |
| `powerUpDropRate` | Frequency of power-up drops | 0.15 |
| `waveSpeedMultiplier` | Enemy speed increase per wave | 1.05 |
| `maxWaves` | Maximum waves before victory | 100 |

To customize, open `index.html` and locate the `gameConfig` object near the top of the script block.

## Compatibility
